# Mixed Messages

A simple JavaScript message generator that prints a new random motivational message every time you run it.

## Run

Use Node.js from the project root:

node mixed-messages/messageGenerator.js

The script will ask for your name and generate a personalized output.

You can also pass your name directly:

node mixed-messages/messageGenerator.js "Alex"

## Browser Version

The portfolio page includes a "Mixed Messages" section with a name field and a Generate button.
Open index.html in a browser and scroll to the section to create random messages instantly.
